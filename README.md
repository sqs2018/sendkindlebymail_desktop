**背景**
    因为小孩天天接收微信群作业，小孩用手机在微信看作业就会玩手机游戏，为了小孩专注力和视力，恰巧家里有一个kindle，因此做了一个将作业分发到Kindle的小工具。
**实现原理是通过邮件将作业发送到kindle。**

**使用步骤**
1.登录你的邮箱，启用POP3/SMTP服务,并增加授权。
